# IoT Environment Monitoring

Небольшой IoT-проект для мониторинга параметров окружающей среды.

## Описание

Система получает данные о температуре, влажности и атмосферном давлении с датчика и публикует их в MQTT-брокер. Spring Boot приложение подписывается на MQTT-топик, обрабатывает входящие сообщения и экспортирует показатели в виде метрик для Prometheus. Grafana используется для визуализации и мониторинга данных в реальном времени.

## Технологии

* MQTT (Mosquitto)
* Java 21
* Spring Boot
* Prometheus
* Grafana
* Docker Compose

## Поток данных

Sensor → MQTT Broker → Spring Boot → Prometheus → Grafana

## Возможности

* Сбор данных с датчика температуры, влажности и давления
* Передача данных через MQTT
* Экспорт метрик в Prometheus
* Визуализация показателей в Grafana
* Контейнеризация с помощью Docker Compose
